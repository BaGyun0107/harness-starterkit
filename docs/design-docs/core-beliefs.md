# Core Beliefs

## Operating Principles

1. **에이전트가 코드를 작성하고, 사람이 방향을 설정한다** — 멀티에이전트 시스템(oh-my-agent)으로 작업을 병렬 실행
2. **모든 변경은 CI만으로 검증 가능해야 한다**
3. **명시적 > 암묵적** — 매직 코드 금지
4. **API 계약 우선** — API 계약을 먼저 정의하고, 프론트/백 각각 구현
