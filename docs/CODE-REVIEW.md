### Every PR

- [ ] TypeScript 컴파일 오류 없음
- [ ] 아키텍처 규칙 준수
- [ ] 새 API는 문서화

## Severity Levels

| Level | Definition | Action |
|-------|-----------|--------|
| BLOCKER | 보안 취약점, 데이터 손실 위험 | 즉시 수정, 머지 불가 |
| MAJOR | 아키텍처 위반, 성능 심각 저하 | 수정 후 머지 |
| MINOR | 코드 스타일, 미미한 개선 | 다음 PR에서 수정 가능 |
| NIT | 선호도 차이, 사소한 제안 | 선택적 반영 |
